# HealthCare ![image](https://i.imgur.com/OhskfS3.png)

## Visão Geral

Visando facilitar e otimizar a anamnese que é feita manualmente nos hemocentros.

## Funcionalidades

- 1- Cadastrar Pessoa
- 2- Listar Doadores
- 3- Excluir Doador
- 4- Atualizar Doador
- 5- Sair
